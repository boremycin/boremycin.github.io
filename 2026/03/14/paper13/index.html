SAR有源物理域攻击和无源物理域攻击的论文笔记


简要介绍

这两篇论文同属于北京化工大学张帆老师课题组, 分别结合的无源角反和有源干扰机进行对抗扰动的物理域实际部署. 需要注意的是这两篇论文都没有实现真正的原型机或提出切实准确的能够用于复现的方法框架. 此外, 两篇论文最大的共同点是都利用了遗传算法(差分进化算法和粒子群优化算法)完成扰动生成的优化过程, 主要借此机会梳理所用的遗传算法的具体细节, 但笔者对这项技术在当前的结合DNN(尤其是可以获取模型梯度信息)的优化场景下的有效性和适用性持强烈的否定态度. 最后, 这两篇论文分别是发表在雷达学报的基于有源干扰机的SAR智能识别对抗方法和发表在JSTAR上的A Physically Realizable Adversarial Attack Method Against SAR Target Recognization Model.

无源干扰

本文的核心思想是将sticker和scatters结合, 分别在目标和背景两部分放置不同形式的扰动, 目标部分放置精细的sticker, 背景部分放置对成像结果影响更明显的角反, 并结合Grad-CAM和PSO算法优化对应的放置位置.

对于sticker部分的优化, 其核心是类似FGSM的梯度上升, 将CAM的热力图结果中对模型预测影响最大的部分作为sticker的部署位置, 此外作者还将扰动通过$tanh(\cdot)$进行了映射. 需要特别注意的是, 这是著名的CW对抗算法的核心思想, 但作者并没有明确的引用和说明.

差分进化算法

差分进化算法(Differental Evolution)是一种元启发(metaheuristic)的方法, 特点是对目标优化问题进行较少的假设, 可以在较大的搜索空间中获取候选结果, 同时带来的显著局限是DE算法无法保证结果是全局最优的. DE算法优化更新的核心步骤是维持一个候选结果的中去, 通过组合已存在的候选结果来获得新的结果, 在这个过程中优化问题本身作为黑盒处理, 仅提供对候选结果质量的查询, 而无需提供梯度信息.

在算法实现过程中, 设$f: \mathbb{R}^n \rightarrow \mathbb{R}$为待优化的目标函数, 优化的目标参数集合可以视为实数向量, 函数的输出是单一实数, 其对应的梯度未知. 设$x \in \mathbb{R}^n$是一个搜索空间中的一个候选结果, DE算法的优化过程大致如下:

  • 设定参数, $NP \geq 4, CR \in [0,1], F \in [0, 2]$
    • NP: 种群大小, 即候选结果的数量
    • CR crossover possibility
    • F 差分权重(differential weight)
    • 通常采用的设定为$NP = 10n, CR = 0.9, F = 0.8$
  • 初始化所有的候选结果$\mathbf{x}$
  • 在满足终止条件(循环次数或最优结果的拟合程度)前重复以下步骤:
    • 从当前种群中随机选择三个候选值$\mathbf{a}, \mathbf{b}, \mathbf{c}$, 这三个值和初始化的$\mathbf{x}$彼此之间互不相同.
    • 从$R \in \{1, …, n\}$中随机选择一个下标, 其中n是搜索空间的维度
    • 计算候选结果的潜在新组合$\mathbf{y} = [y_1,…, y_n]$, 计算方法如下:
      • 对于$i \in \{1,…, n \}$, 获取对应的服从均匀分布的随机数$r_i \sim U(0,1)$
      • 如果$r_i < CR$ 或者$i = R$, 设定$y_i = a_i + F \times (b_i - c_i)$ 否则 $y_i = x_i$, 其中R的作用是对更新的候选结果至少保留一个变异参数
    • 如果$f(\mathbf{y} \leq \mathbf{x})$, 用$\mathbf{y}$替换$\mathbf{x}$
  • 从更新结束后的所有候选结果中选出最优结果.

粒子群算法

粒子群优化算法（Particle Swarm Optimization, PSO）是一种模拟鸟群觅食行为的元启发（metaheuristic）方法。其核心逻辑在于通过个体间的协作与信息共享来引导整个种群在较大的搜索空间内寻找候选解。与DE算法类似，PSO对优化问题本身进行黑盒处理，不依赖目标函数的梯度信息，但也面临着在高维非凸空间中容易陷入局部最优且无法保证全局收敛的显著局限。PSO维持一个由多个“粒子”组成的种群，每个粒子代表一个候选结果，并通过动态调整位置与速度来演化。

在算法实现过程中，设 $f: \mathbb{R}^n \rightarrow \mathbb{R}$ 为待优化的目标函数。每个粒子包含两个实数向量：位置 $\mathbf{x} \in \mathbb{R}^n$ 和速度 $\mathbf{v} \in \mathbb{R}^n$。此外，算法需记录粒子自身的历史最优位置 $\mathbf{pbest}$ 以及整个种群的全局最优位置 $\mathbf{gbest}$。PSO的优化过程大致如下：

  • 设定参数，$N \geq 20, w \in [0, 1], c_1, c_2 \in [0, 2]$
    • N: 种群大小（粒子数量）
    • $w$: 惯性权重（Inertia Weight），用于控制前一时刻速度对当前的影响
    • $c_1, c_2$: 加速常数或学习因子（Cognitive and Social Factors）
    • 通常设定为 $w=0.8, c_1=1.5, c_2=1.5$
  • 初始化所有粒子的位置 $\mathbf{x}$ 和速度 $\mathbf{v}$，并初始化其 $\mathbf{pbest}$ 和 $\mathbf{gbest}$
  • 在满足终止条件前重复以下步骤：
    • 对于每个粒子，生成两个独立的随机向量 $r_1, r_2 \sim U(0, 1)$
    • 更新粒子的速度：$\mathbf{v} = w\mathbf{v} + c_1 r_1 (\mathbf{pbest} - \mathbf{x}) + c_2 r_2 (\mathbf{gbest} - \mathbf{x})$
    • 更新粒子的位置：$\mathbf{x} = \mathbf{x} + \mathbf{v}$，通常需对位置和速度进行边界裁剪（Clipping）
    • 计算新的适应度 $f(\mathbf{x})$：
      • 如果 $f(\mathbf{x}) < f(\mathbf{pbest})$，更新个体最优 $\mathbf{pbest} = \mathbf{x}$
      • 如果 $f(\mathbf{x}) < f(\mathbf{gbest})$，更新全局最优 $\mathbf{gbest} = \mathbf{x}$
    • 从更新结束后的全局最优位置 $\mathbf{gbest}$ 中得出最终结果。
